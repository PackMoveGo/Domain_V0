## Version Numbering

- **Major.Minor.Patch** (e.g., 1.0.1)
  - **Major**: Breaking changes, major feature additions
  - **Minor**: New features, backward compatible
  - **Patch**: Bug fixes, minor improvements

## Release Types

- **[Unreleased]**: Changes in development, not yet released
- **[Version]**: Released versions with date
- **Added**: New features
- **Changed**: Changes in existing functionality
- **Deprecated**: Soon-to-be removed features
- **Removed**: Removed features
- **Fixed**: Bug fixes
- **Security**: Security-related changes

## Contributing

When adding entries to this changelog, please follow these guidelines:

1. Use clear, concise language
2. Group changes by type (Added, Changed, Fixed, etc.)
3. Include relevant details for significant changes
4. Use present tense ("Add" not "Added")
5. Reference issues and pull requests when applicable
6. Include breaking changes prominently
7. Add security-related changes to the Security section
